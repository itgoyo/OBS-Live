# OBS-Live

### 麦克风出声模式

直播间音频设置 VoiceMeeter 

Input1-> VoiceMeeter Aux Output(mme vb-audio)

输出只有一个A1 DELL显示器

OBS音频选择

桌面音频-> 扬声器（VB-Audio VoiceMeeter AUX VAIO）

麦克风/Aux-> Analogue 1/2(audient iD14) 麦克风单声道的解决方式：编辑-高级音频属性-单声道的钩打上就可以了

### 不是用麦克风的情况

不需要开启机架的开关
